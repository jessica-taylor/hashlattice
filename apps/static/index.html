Welcome to HashLattice

Welcome to the project page for HashLattice!

HashLattice is a distributed network which, as the title suggests, is based on hash codes and lattices. Specifically, we allow users to specify pages by their hash code for retrieval from peers in the network. The "lattice" in HashLattice refers to how we allow people putting pages into the network to dynamically update their content via a special "merge" function, specified by the page creator, which functions as a lattice in that the merge function takes two values and returns either the better value of the two or some combination of the values, like a least upper bound operator in a lattice.
